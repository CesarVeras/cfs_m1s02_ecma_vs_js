[M1S02] Ex 1 - Qual a relação entre ECMAScript e JavaScript

Com suas palavras descreva a relação entre ECMAScript e JavaScript.

Escreva da maneira que você considerar mais fácil a diferença entre as linguagens Javascript e ECMAScript.

JavaScript é uma linguagem de programação.

Já o ECMAScript é o conjunto de padrões, normas e funções de determinada versão do javascript. O ECMA 6 por exemplo, trouxe melhorias para o JavaScript como: let, const, funções de array, funções matemáticas, etc.
